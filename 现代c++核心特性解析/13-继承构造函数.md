## 继承构造函数
---
#### 1. c++11 继承构造函数的使用<br>
    c++11 子类可以用利用基类的构造函数，这样减少了代码量
```c++
class Base
{
 public:
    Base(int n)

    int age;
};

class Derived : public Base
{
 public:
    using Base::Base;
};
```

需要注意的是
1. 派生类是隐式继承基类构造函数，只有在程序中使用，才会生成继承构造函数代码
2. 派生类不会继承基类的默认构造函数和复制构造函数
3. 派生类对继承构造函数进行重载，那么不会继承该继承构造函数
4. 派生类同时继承相同的构造函数会导致编译失败
5. 基类构造函数不能为私有
